# 有关 Azure OpenAI 的文档

# 申请经历
- Azure portal 可以找到申请入口，https://aka.ms/oai/access， 填表，可以找个 @microsoft.com 的朋友帮忙
- 用了我的公司，用了域名支付证明
- 等了约一个约，补了一次材料，终于下来了，和 Azure 订阅绑定

# 好处
- 不用翻墙
- Azure结算费用，可以绑定国内信用卡
- API应该更稳定，更适宜企业使用
- 可以换部署区，现在只有 美东 和 欧洲可选
