# mac 安装 java
- (没成功) brew
  - brew install openjdk@8
  - https://stackoverflow.com/a/65957215/4345230
- (成功) sdkman
  - ```curl -s "https://get.sdkman.io" | bash```
  - ```sdk list java ```
  - ```sdk install java 8.0.322-tem ```
  - 安装完成后，自动设置好 Path, 很方便
  - 在inteliJ里面，可以查到具体版本和current版本的sdk，很方便
  - https://stackoverflow.com/a/65957215/4345230
